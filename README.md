# Radiant Dawn Asset Repository

A community-driven resource for **Fire Emblem: Radiant Dawn** modding, ROM hacking, reverse engineering, and fan projects.

This repository serves as a centralized collection of assets, code, and development resources created or gathered for the Radiant Dawn modding community. The goal is to make high-quality resources easily accessible while respecting the work of individual contributors.

---

# Repository Contents

This repository may contain, but is not limited to:

* Character portraits
* UI graphics and icons
* Affinity icons
* Item and weapon graphics
* Skill and scroll assets
* Emotes and miscellaneous artwork
* Scripts and event resources
* Custom code and gameplay systems
* Reverse-engineering resources
* Documentation and development tools

New resources may be added over time as the project and community continue to grow.

---

# Usage Policy

Unless otherwise stated, all resources contained within this repository are free to use in both personal and public projects.

You may:

* Use assets in ROM hacks and fan projects
* Include assets in public releases
* Redistribute assets as part of your project
* Build upon existing resources where permitted
* Learn from and reference included code

The intention of this repository is to encourage collaboration and support the continued growth of the Radiant Dawn modding community.

---

# Attribution & Creator Tags

Most assets in this repository do **not** require attribution.

However, files that contain a creator tag in their filename must be credited when used.

Examples:

```text
Almedha (Young) {ATMachine}.png
Elena (Long Hair) {ATMachine}.png
Bryce (Young) {ATMachine}.png
```

In these examples, the creator is identified as **ATMachine**.

When using assets that contain a creator tag, please provide appropriate credit to the creator listed within the brackets.

Example:

```text
Portrait by ATMachine
```

If multiple creators are listed, all listed creators should be credited.

---

# Modification Rules

Assets without a creator tag may be freely modified.

Assets containing a creator tag (`{Name}`) may be used in projects, but may **not** be edited, altered, recolored, expanded, spliced, redrawn, AI-modified, or otherwise modified without explicit permission from the original creator.

This restriction includes, but is not limited to:

* Recolors
* Portrait edits
* Splices
* Derivative works
* AI-generated edits
* Touch-ups
* Asset expansions
* Custom variations

If you wish to modify a tagged asset, please obtain permission from the creator first.

# Contributing

Contributions are welcome.

If you wish to contribute assets, please follow these guidelines:

### Assets Requiring Attribution

If you would like users to credit you when using your work, include your creator name within braces:

```text
Portrait Name {Creator}.png
```

Example:

```text
Elena (Long Hair) {ATMachine}.png
```

### Assets Without Attribution Requirements

If you do not require attribution, simply omit the creator tag from the filename.

### Contributor Responsibility

By submitting content to this repository, contributors affirm that they have the right to distribute the submitted work and grant usage permissions according to this repository's policies.
